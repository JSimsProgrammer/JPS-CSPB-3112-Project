Key Takeaways

This week I read about vectors. Now, first thing is first: When I am talking about vectors, I am talking about Euclidean Vectors, which are definited in The Nature of Code as "an entity that has both magnitude and direction." So, unless stated otherwise, these vectors are going to have only two elements: Magnitude and direction. These Euclidean vectors have many uses when coding forces that occur in the natural world. The book uses the following example: You write a simple program to have a ball bounce around a box. You could either use four variables to dictate the balls movement (x, y, xspeed, yspeed ), or you could use a two vector class objects (Location and Speed). While these vectors are not not technically using magnitude and direction, they do each only have two elements.

  • Vector Multiplication and Division aka Scaling
  • Vector Mangitude
